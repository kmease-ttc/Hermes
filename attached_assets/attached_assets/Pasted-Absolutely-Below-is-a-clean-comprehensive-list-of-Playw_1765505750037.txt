1. Core Funnel & Site Generation (Highest Priority)

1.1 Generate Free Site (Anonymous)

Goal: Ensure the core value proposition never breaks.
	•	Load landing page
	•	Click “Generate Free Site”
	•	Fill required fields
	•	Optionally fill phone/email
	•	Submit
	•	Assert:
	•	preview page loads
	•	no console errors
	•	preview URL renders content
	•	CTA “Publish My Site” exists

Fail if:
	•	generation stalls
	•	blank preview
	•	JS errors
	•	missing CTA
