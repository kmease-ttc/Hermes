1. Technical SEO Review (Full Checklist)

A. Indexability & Crawlability

Your developer needs to verify:

1. Robots.txt
	•	Ensure it exists at arclo.pro/robots.txt
	•	Allow all essential folders
	•	Block system folders (e.g., /api/, /admin/, /internal/ if applicable)
	•	Include your sitemap:
Sitemap: https://arclo.pro/sitemap.xml

2. Sitemap
	•	Auto-generate with every new landing page
	•	Separate sitemaps:
	•	Main pages
	•	Blog/SEO content pages
	•	Dynamic AI-generated landing pages
	•	Update daily or live

3. JavaScript Rendering

Arclo loads via client-side JS. You should ensure:
	•	Pre-render or SSR using Next.js or Nuxt (best for SEO)
	•	Or add Prerender.io (fastest fix)
	•	Or use Cloudflare Workers HTML caching
If Google can’t see content until JS loads, you will get zero indexation.

4. Canonicals
	•	Every URL must include <link rel="canonical" />
	•	Especially critical if AI generates many pages

5. URL Structure
	•	Use lowercase, hyphens, descriptive paths:
	•	/seo-automation/
	•	/ai-seo-tools/
	•	/local-seo/
Avoid:
	•	/page?id=38jd82
	•	/aiSEOtool

6. Duplicate Content Control

Because Arclo can generate hundreds of pages:
	•	Add MD5 hash comparison before publishing
	•	Use canonicals for variations
	•	Prevent productized pages from rewriting the same concepts endlessly

2. On-Page SEO Requirements

A. Title Tags

Every page must follow a strict format:

Primary Keyword | Brand Secondary Value

Examples:
	•	“AI SEO Automation Software | Arclo”
	•	“Automated Website Builder | Arclo”
	•	“SEO Insights Dashboard | Arclo”

Character length: 55–60

B. Meta Descriptions

Formula: Outcome + Differentiator + CTA

Example:
“Automate your SEO with AI. Arclo builds pages, optimizes content, and tracks rankings in real time. Start free.”

C. H1/H2/H3 Structure
	•	1 H1 only
	•	H2 for major sections
	•	H3 for nested items

D. Image SEO

Every image must include:
	•	alt
	•	compressed WebP
	•	filename that includes the target keyword

3. Performance & Core Web Vitals

What your dev must fix:
	•	Large Contentful Paint < 1.8s
	•	Preload hero image
	•	Reduce DOM depth
	•	CLS = 0.00–0.05
	•	Set fixed height on images
	•	Total Blocking Time < 150ms
	•	Split JavaScript bundles
	•	Lazy-load noncritical JS

Use:
	•	Cloudflare Polish
	•	Cloudflare APO
	•	Next.js ISR if you migrate

4. Schema / Structured Data

Add schema via JSON-LD:

For main site:
	•	WebSite
	•	Organization
	•	BreadcrumbList
	•	Product schema for each plan
	•	FAQ schema for FAQ section
	•	Review schema (when you have real reviews)

For blog/SEO content:
	•	Article schema
	•	Author schema
	•	FAQ schema inside posts

This alone generates rich snippets.
