# 1. **Brand Colors**

## **Primary Palette**

| Purpose             | Color     | Use                                    |
| ------------------- | --------- | -------------------------------------- |
| **Primary Indigo**  | `#4F46E5` | Primary buttons, icons, strong accents |
| **Electric Purple** | `#7C3AED` | Gradients, icon glows, highlights      |
| **Sky Blue**        | `#38BDF8` | Accent gradient hues, soft highlights  |
| **Deep Sapphire**   | `#181857` | Dark sections, backgrounds             |

These act as your **core AI gradient**:

### **Primary Gradient (for buttons, tags, highlights)**

```
bg-gradient-to-r from-[#4F46E5] via-[#7C3AED] to-[#38BDF8]
```

---

## **Neutral Palette**

| Purpose       | Color     | Use                                      |
| ------------- | --------- | ---------------------------------------- |
| **Slate 900** | `#0F172A` | Headings, dark text                      |
| **Slate 700** | `#374151` | Body text                                |
| **Slate 500** | `#64748B` | Secondary text                           |
| **Slate 200** | `#E5E7EB` | Borders, dividers                        |
| **Slate 100** | `#F1F5F9` | Light cards, backgrounds                 |
| **White**     | `#FFFFFF` | Hero card, pricing cards, content blocks |

---

## **AI Glow Colors**

Used for gradients behind mockups, stat sections, floating chips.

* **Glow Purple:** `rgba(124, 58, 237, 0.35)`
* **Glow Blue:** `rgba(79, 70, 229, 0.25)`
* **Glow Pink:** `rgba(244, 114, 182, 0.25)`

These should always be blurred heavily (`blur-3xl` or `blur-[80px]`).

# 2. **Typography**

## **Headings: Manrope**

```
font-family: 'Manrope', sans-serif;
```

* Rounded but strong
* Great for large, premium SaaS headings
* Use for **all page titles and hero text**

**Usage:**

* `text-5xl font-extrabold tracking-tight` ‚Üí Hero
* `text-3xl font-bold` ‚Üí Section titles
* `text-xl font-semibold` ‚Üí Subheaders

---

## **Body: Inter**

```
font-family: 'Inter', sans-serif;
```

**Usage:**

* `text-base text-slate-600` for paragraphs
* `text-sm text-slate-500` for microcopy
* `text-lg text-slate-700` for important description text

# 5. **Motion Guidelines**

Motion should feel subtle, premium, not playful.

### Hero mockup:

* `animate-float-slow` (3‚Äì6px vertical float)

### Floating tags:

* `transition-all duration-300 ease-out hover:scale-105`

### Cards (features, pricing):

* Slight lift: `hover:-translate-y-1`
* Subtle brighten: `hover:shadow-xl`
