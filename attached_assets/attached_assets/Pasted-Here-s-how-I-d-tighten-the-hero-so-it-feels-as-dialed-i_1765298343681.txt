## 1. Background + Hero Card

Right now the hero card looks good but a little flat compared to the upgraded sections.

Use the same light gradient + glow language you’ve used elsewhere:

**Section wrapper**

```tsx
<section className="relative bg-gradient-to-b from-white via-slate-50 to-slate-100 py-20 md:py-24">
  <div className="max-w-6xl mx-auto px-4 md:px-8">
    {/* hero card */}
  </div>
</section>
```

**Glow behind the card** (subtle, like the stats section):

```tsx
<div className="pointer-events-none absolute -left-24 top-0 h-64 w-64 rounded-full bg-indigo-300/20 blur-3xl" />
<div className="pointer-events-none absolute right-[-40px] top-32 h-72 w-72 rounded-full bg-purple-400/14 blur-3xl" />
```

**Hero card**

```tsx
<div className="relative rounded-[32px] bg-white/95 backdrop-blur-xl shadow-[0_24px_70px_rgba(15,23,42,0.18)] px-8 py-12 md:px-12 md:py-14 flex flex-col lg:flex-row gap-10 lg:items-center">
  {/* left + right */}
</div>
```

That gives you the same “glass + depth” vibe as the dark stats and pricing.

Style the four tags to look like mini glass chips, consistent with your feature icons and stats:

```tsx

  "absolute inline-flex items-center gap-2 rounded-full bg-white/95 backdrop-blur-xl px-4 py-2 text-xs font-medium text-slate-700 shadow-[0_10px_30px_rgba(15,23,42,0.18)] border border-slate-200/80";

<span className={`${tagBase} -top-6 left-4`}>
  Site Improved Daily
  <span className="text-emerald-500">✓</span>
</span>

<span className={`${tagBase} -top-4 right-0`}>
  Content Written
  <span className="text-emerald-500">✓</span>
</span>

<span className={`${tagBase} -bottom-6 left-0`}>
  Pages Generated
  <span className="text-emerald-500">✓</span>
</span>

<span className={`${tagBase} -bottom-4 right-4`}>
  SEO Issues Fixed
  <span className="text-emerald-500">✓</span>
</span>
```

Small extra polish: give them a slight hover interaction:

```tsx
className={`${tagBase} hover:-translate-y-0.5 hover:shadow-[0_14px_36px_rgba(15,23,42,0.26)] transition-transform duration-200`}
```

Now the tags visually match the glass stats and feature cards instead of feeling like a different system.

## 5. Optional Micro-Interaction on Scroll

To tie it all together with the header:

* Fade the hero card in from 8–12px below on load
* Slight stagger between left column and mockup

If you’re already using Framer Motion:

```tsx
<motion.div
  initial={{ opacity: 0, y: 16 }}
  animate={{ opacity: 1, y: 0 }}
  transition={{ duration: 0.5, ease: "easeOut" }}
>
  {/* hero card */}
</motion.div>
```
