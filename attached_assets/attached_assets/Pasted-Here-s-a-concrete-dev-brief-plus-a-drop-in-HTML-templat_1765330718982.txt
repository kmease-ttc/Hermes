Here’s a concrete “dev brief” plus a drop-in HTML template they can start from.

---

### 1. Goal

Redesign the “Your Website Preview is Ready!” email so it:

* Looks like the Arclo marketing site (modern SaaS feel)
* Clearly explains what happened and what to do next
* Works well on mobile and in common email clients

---

### 2. Implementation steps

1. **Find the existing template**

   * Locate the function that sends this email (e.g. `sendSitePreviewEmail`, `sendPreviewReadyEmail`).
   * It will either:

     * Build a string of HTML directly, or
     * Load a template file (e.g. in `shared/emailTemplates` or `server/emails`).

2. **Replace the current HTML body with the template below**

   * Keep the subject line logic as-is (or update it to match: `Your Arclo website preview is ready ✨`).
   * Inject dynamic values:

     * `{{userName}}` (optional)
     * `{{siteName}}` or `{{businessName}}`
     * `{{previewUrl}}`
     * `{{expiresAt}}` (e.g. “7 days”)

3. **Make sure the sender matches the brand**

   * `from: "Arclo <no-reply@arclo.pro>"` (or whatever verified sender you’re using in SendGrid).
   * Reply-to: `support@arclo.pro`.

4. **Test**

   * Send test emails to Gmail (web + mobile), Apple Mail, and Outlook.
   * Verify:

     * Button works.
     * Layout is centered and responsive.
     * Text is legible in dark mode (no very light gray on white).

---
