GLOBAL (APPLIES TO EVERY PAGE)

These should be verified once, then rechecked everywhere.
	•	Ensure consistent typography, spacing, and color usage
	•	Remove all placeholder copy, images, or default states
	•	Confirm mobile responsiveness (no overlap, truncation, or hidden CTAs)
	•	Verify page loads < 3s on mobile
	•	Confirm analytics fires on every page
	•	Ensure consistent header + footer across site
	•	Add meta title + description for every indexable page
	•	Add favicon (correct asset, not default)
	•	Confirm HTTPS only (no mixed content)
	•	Validate canonical tags
	•	Ensure no console errors on load
	•	Confirm accessibility basics (button contrast, alt text, labels)
