Yep — on that `/how-it-works` page the **text is basically invisible** because it’s still using **“dark-theme” text styles** (very light/low-opacity text) on a **light background**.

You fix it by changing the marketing pages to use **dark text tokens** (Slate 900/700) and removing any `opacity-*` / `text-white/xx` classes.

## What’s wrong (what I see)

* The headline and paragraph are rendered in **very light gray** (looks like `text-slate-200/300` or `text-white/40`), so on a white background it washes out.
* Step number + body copy are also too light.
* This is almost always coming from a shared “marketing-theme” class or a component variant that still assumes dark mode.

## Fix (do this in your code)

### 1) Update the marketing layout wrapper

Find the wrapper div in your marketing layout (likely `client/src/components/layout/MarketingLayout.tsx` or similar) and ensure it sets the base text color:

* Background: `bg-slate-50` (or `bg-white`)
* Base text: `text-slate-900`

Example:

```tsx
<div className="min-h-screen bg-slate-50 text-slate-900">
  ...
</div>
```

### 2) Fix the hero + section text classes

On this page component (likely `client/src/pages/HowItWorks.tsx` or `client/src/pages/marketing/how-it-works.tsx`):

Replace any of these patterns:

* `text-white/60`, `text-white/70`
* `text-slate-200`, `text-slate-300`, `text-slate-400` for body text
* `opacity-60`, `opacity-70` on containers

With:

**Headline**

* `text-slate-950` (or `text-slate-900`)

**Body**

* `text-slate-700`

**Muted**

* `text-slate-500`

Example:

```tsx
<h1 className="text-4xl font-semibold tracking-tight text-slate-950">
  From diagnosis to deployment in three simple steps.
</h1>

<p className="mt-4 text-lg text-slate-700">
  Arclo runs the same comprehensive checks that experienced SEO professionals use...
</p>
```

### 3) Fix nav link colors (top right)

Your nav links are also too faint. Use:

* Default: `text-slate-700`
* Hover: `text-slate-950`

Example:

```tsx
<a className="text-slate-700 hover:text-slate-950" ...>How It Works</a>
```

### 4) Fix “step” number and checklist icons

Step circle:

* Background: `bg-violet-600` (or your chosen accent)
* Text: `text-white`

Checklist icons:

* `text-emerald-600`

Body text next to it:

* `text-slate-700`
