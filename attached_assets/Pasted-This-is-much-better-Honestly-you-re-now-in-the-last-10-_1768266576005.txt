## 2. Label overlap (yes, it needs fixing — here’s how)

### Where it’s happening

From what I can see, overlap is coming from:

* Status pills (“Checked”, “Needs Attention”, “Limited Scan”)
* Card titles + badges competing horizontally
* Some cards not reserving vertical space for labels

### Root cause

The header row of cards is trying to do **too much in one line**.

### Fix (clean and simple)

**Move to a strict two-row card header pattern.**

#### Card header layout (lock this in):

**Row 1**

* Icon
* Title (single line, truncates if needed)

**Row 2**

* Status pills (wrap allowed)
* Smaller font than title
* Consistent spacing

Do NOT allow pills to sit on the same baseline as the title anymore.

This one change will:

* Eliminate overlap
* Improve scanability
* Make cards feel calmer

### Implementation detail

* Give the card header a fixed min-height
* Use `flex-col` for header
* Allow pills to wrap (`flex-wrap`)
* Slightly reduce pill font size (they’re metadata, not headlines)
